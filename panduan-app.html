📱 Pengenalan Aplikasi

Aplikasi Pencarian Data Penduduk adalah sistem informasi berbasis web yang dirancang untuk memudahkan pengelolaan dan pencarian data penduduk secara digital. Aplikasi ini menyediakan antarmuka yang user-friendly dengan berbagai fitur pencarian canggih.

💡

Keunggulan Aplikasi:

  • Akses cepat ke data penduduk dari mana saja
  • Pencarian multi-kriteria (nama, NIK, alamat)
  • Filter berdasarkan dusun
  • Export data ke format PDF
  • Berbagi data via WhatsApp
  • Tampilan responsif untuk semua perangkat
  • Sistem keamanan dengan password admin

🔍 Cara Mencari Data Penduduk

1

Ketik Kata Kunci

Masukkan kata kunci pencarian di kolom pencarian. Anda bisa mencari berdasarkan:

  • 👤 Nama: "Ahmad", "Fatimah", dll
  • 🆔 NIK: 16 digit nomor induk
  • 👨‍👩‍👧‍👦 No. KK: Nomor Kartu Keluarga
  • 🏠 Alamat: Nama jalan, gang, RT/RW
  • 🏘️ Dusun: Nama dusun
2

Klik Tombol Cari (Opsional)

Tekan tombol "🔍 Cari" atau cukup ketik dan aplikasi akan mencari otomatis.

💡 Tips Pencarian: Aplikasi akan otomatis mencari saat Anda mengetik (Auto-search). Tidak perlu tekan tombol cari setiap kali!

3

Lihat Hasil Pencarian

Hasil pencarian akan ditampilkan dalam bentuk kartu data dengan informasi ringkas.

📊 Informasi yang ditampilkan:

  • Foto penduduk (jika tersedia)
  • Nama lengkap dan NIK
  • Data identitas utama (KK, jenis kelamin, tempat lahir, dll)
  • Nomor urut data
4

Klik untuk Lihat Detail

Klik pada kartu data untuk membuka modal detail lengkap penduduk.

💡

Tips Pencarian Efektif:

  • • Gunakan kata kunci spesifik untuk hasil lebih akurat
  • • Tidak perlu menulis kata kunci lengkap (sebagian kata juga bisa)
  • • Pencarian tidak case-sensitive (huruf besar/kecil sama saja)
  • • Kombinasikan dengan filter dusun untuk hasil lebih fokus
  • • Hapus kata kunci untuk menampilkan semua data kembali

🏘️ Filter Berdasarkan Dusun

Filter dusun memudahkan Anda untuk melihat data penduduk per wilayah tertentu tanpa perlu mencari manual.

1

Temukan Bagian Filter Dusun

Di bagian atas halaman, Anda akan melihat deretan tombol dusun dengan ikon 🏘️

🏘️

Tombol Filter: Semua Dusun | Dusun A | Dusun B | Dusun C

2

Pilih Dusun

Klik salah satu tombol dusun untuk memfilter data. Tombol yang aktif akan berubah warna menjadi merah-emas.

🎯 Jenis Filter:

  • Semua Dusun: Menampilkan seluruh data penduduk
  • Dusun Tertentu: Hanya menampilkan data dari dusun yang dipilih
3

Lihat Statistik

Badge statistik akan menunjukkan nama dusun yang aktif dan jumlah data yang ditampilkan.

📊 Badge Statistik:

  • • Total: Jumlah seluruh data di database
  • • Ditampilkan: Jumlah data sesuai filter aktif
  • • Dusun: Nama dusun yang sedang difilter
4

Kombinasi dengan Pencarian

Anda bisa mengkombinasikan filter dusun dengan pencarian kata kunci untuk hasil yang lebih spesifik.

Contoh Penggunaan:

1. Pilih "Dusun A"

2. Ketik "Ahmad" di kolom pencarian

→ Hasil: Semua data dengan nama "Ahmad" di Dusun A

5

Reset Filter

Untuk kembali melihat semua data, klik tombol "🏘️ Semua Dusun"

📋 Melihat Detail Data Penduduk

1

Klik Kartu Data

Klik pada kartu data penduduk yang ingin Anda lihat detailnya. Kartu akan menampilkan teks "Klik untuk lihat detail lengkap 👁️"

2

Modal Detail Terbuka

Jendela popup (modal) akan muncul menampilkan informasi lengkap yang terbagi dalam beberapa bagian:

👤 Data Identitas

Foto, Nama Lengkap, NIK, No. KK, Jenis Kelamin

💡 Tombol "📋 Salin" tersedia untuk menyalin NIK dan No. KK dengan cepat

🎂 Data Kelahiran

Tempat Lahir, Tanggal Lahir, Agama, Umur

🏠 Alamat & Lokasi

Dusun, Suku, Alamat Lengkap

ℹ️ Informasi Lainnya

Pendidikan, Pekerjaan, Status Kawin, Kewarganegaraan, Status

👪 Data Orang Tua

Nama Ayah, NIK Ayah, Nama Ibu, NIK Ibu (jika tersedia)

3

Fitur dalam Modal Detail

📋

Salin NIK / No. KK

Klik tombol "📋 Salin" di samping NIK atau No. KK untuk menyalin ke clipboard. Notifikasi akan muncul saat berhasil.

📱

Bagikan ke WhatsApp

Klik tombol "📱 Bagikan ke WhatsApp" di bagian bawah modal untuk membagikan data dalam format terstruktur.

👨‍💼

Info Administrator

Bagian footer menampilkan informasi administrator (Syam Digital) yang mengelola data.

4

Tutup Modal

Untuk menutup modal detail, Anda bisa:

  • Klik tombol × di pojok kanan atas
  • Klik di area gelap di luar modal
  • Tekan tombol ESC pada keyboard

📄 Export Data ke PDF

Fitur export PDF memungkinkan Anda mencetak atau menyimpan data penduduk dalam format dokumen PDF yang rapi dan profesional.

1

Klik Tombol Export PDF

Temukan dan klik tombol "📄 Export PDF" di bagian atas halaman (di samping tombol "📊 Buka Spreadsheet")

2

Pilih Opsi Export

Menu popup akan muncul dengan pilihan data yang ingin di-export:

🏘️ Semua Dusun
xxx data

Export seluruh data penduduk dari semua dusun

🏘️ Dusun A
xx data

Export hanya data dari Dusun A

... dan seterusnya untuk setiap dusun yang tersedia

3

Klik Opsi yang Diinginkan

Pilih salah satu opsi dengan mengklik. Tombol akan berubah menjadi "⏳ Membuat PDF..."

4

PDF Otomatis Terunduh

File PDF akan otomatis dibuat dan terunduh ke perangkat Anda dengan format nama:

Data_Penduduk_Semua_[timestamp].pdf

atau

Data_Penduduk_[NamaDusun]_[timestamp].pdf

📋

Isi PDF yang Di-Export:

  • • Header: Judul laporan dan tanggal export
  • • Statistik: Total jumlah data yang di-export
  • • Tabel Data: No, Nama, NIK, Jenis Kelamin, Tempat Lahir, Tanggal Lahir, Dusun, Alamat
  • • Footer: Nomor halaman di setiap lembar
  • • Format: Ukuran kertas A4 dengan orientasi Portrait
⚠️

Catatan Penting:

  • • Pastikan browser Anda mengizinkan download otomatis
  • • Proses pembuatan PDF mungkin membutuhkan waktu untuk data yang besar
  • • File PDF dapat langsung dibuka, dicetak, atau dibagikan
  • • Untuk data sangat besar, pertimbangkan export per dusun

❓ Pertanyaan yang Sering Diajukan (FAQ)

🔐 Bagaimana jika saya lupa password?

Hubungi administrator sistem (Syam Digital) melalui tombol WhatsApp di header aplikasi untuk mendapatkan bantuan reset password.

🔄 Kenapa data tidak update setelah diedit di Spreadsheet?

Setelah mengedit data di Google Spreadsheet, refresh halaman aplikasi (tekan F5 atau Ctrl+R) untuk memuat data terbaru.

📷 Kenapa foto tidak muncul?

Pastikan URL foto di Spreadsheet valid dan dapat diakses. Foto dari Google Drive harus memiliki izin akses publik. Jika foto tetap tidak muncul, akan ditampilkan placeholder icon.

🔍 Kenapa hasil pencarian tidak ada padahal data ada?

Periksa apakah filter dusun aktif. Jika ya, data mungkin berada di dusun lain. Klik "Semua Dusun" untuk mencari di seluruh data.

📱 Apakah bisa diakses dari HP?

Ya, aplikasi fully responsive dan dapat diakses dari smartphone Android maupun iOS melalui browser (Chrome, Safari, Firefox).

📄 Berapa lama waktu untuk export PDF data besar?

Waktu export tergantung jumlah data dan spesifikasi perangkat. Untuk ratusan data mungkin membutuhkan 5-10 detik. Untuk ribuan data bisa lebih lama.

💾 Apakah data tersimpan di perangkat saya?

Tidak, semua data tersimpan di Google Spreadsheet cloud. Aplikasi hanya menampilkan data dari sumber tersebut secara real-time.

🌐 Apakah perlu koneksi internet?

Ya, koneksi internet diperlukan untuk mengakses data dari Google Spreadsheet dan menggunakan semua fitur aplikasi.

🔒 Seberapa aman data penduduk?

Aplikasi dilindungi dengan password admin. Hanya pengguna yang memiliki password yang dapat mengakses data. Google Spreadsheet juga memiliki sistem keamanan tersendiri.

✏️ Bagaimana cara menambah atau mengedit data?

Klik tombol "📊 Buka Spreadsheet" untuk mengakses Google Sheets. Di sana Anda bisa menambah, mengedit, atau menghapus data langsung.

📊 Berapa maksimal data yang bisa ditampilkan?

Aplikasi dapat menampilkan ribuan data dengan performa baik. Namun untuk pengalaman terbaik, gunakan filter dusun dan pencarian untuk membatasi hasil.

🎨 Apakah tampilan bisa diubah?

Tampilan sudah dioptimalkan untuk pengalaman terbaik. Jika ada permintaan khusus, hubungi developer (Syam Digital) untuk kustomisasi.
